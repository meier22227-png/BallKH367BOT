# Football Pitch & Rules Master Bot

An interactive Telegram bot built with Node.js and Telegraf for t.me/BallKH367BOT. Operating completely offline without external APIs, it helps football players, amateur referees, and tournament organizers check official pitch standards and match rules instantly.

## Features
- Pitch Dimensions: Official metrics for 11v11, 7v7, and 5v5 (Futsal) surfaces.
- Rule Decrypter: Simplified text explanations of complex Offside and Handball criteria.
- VAR Protocols: Breakdown of when the Video Assistant Referee can intervene.

## Deployment on Railway
1. Push this repository to GitHub.
2. Link your repository to Railway.app.
3. Create a config environment variable named BOT_TOKEN containing your BotFather token.
4. Deploy!
